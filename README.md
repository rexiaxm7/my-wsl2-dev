# How to Create My WSL2 Dev.
## 1. Download and unzip.
```
curl -LO https://github.com/rexiaxm7/my-wsl2-dev/archive/master.zip
unzip master.zip
cd my-wsl2-dev-master
```
## 2. Update and upgrage.
```
sudo apt update -y
sudo apt upgrade -y
```

## 3. Install Homebrew.
```
sh homebrew/install.sh
```

## 4. Install Zsh
```
sh zsh/install.sh
```

## 5. Install Zinit
```
sh zinit/install.sh
```
